# Hanoi Tower

## How to play
```plaintext
python game.py
```

The game will be like:
```plaintext
  *   |       |       | 
 ***  |       |       | 
***** |       |       | 
  0       1       2
Select disk: 0
Move disk: 2
      |       |       | 
 ***  |       |       | 
***** |       |   *   | 
  0       1       2
Select disk: 
... 

```


To adjust the number of disks, use argument `num_disk`.
```
python game.py --num_disk 5
```
